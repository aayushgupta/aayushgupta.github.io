# Game of Life

**Java Files**

1. GameOfLife.java -> Contains the rules to run against the seed universe and create the new generation.
2. GameOfLifeRunner.java -> Contains the main method to start the game.
3. GameBoardNotInitializedException.java -> Custome exception in case the game cannot be started for any reason. 



**How to Run:**
GameOfLifeRunner class provides the main method, to provide the seed universe (2-d array), and we can provide the number for generations for which we want to run 
the game. The state of universe after each generation will be displayed on the console.
